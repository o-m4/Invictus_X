# Smart Civic Service Platform

AI-powered platform for citizen complaints and government service management, designed to improve transparency, accountability, and response time in public service delivery.

This project was developed for the Digital Democracy Hackathon.

**Problem Statement -**

Citizens often face delays and lack transparency when reporting issues or requesting government services. Existing systems provide limited tracking, weak accountability, and slow coordination between departments.

This results in:

- Unresolved complaints.

- Inefficient service delivery.

- Reduced public trust in governance systems.

**Solution -**

Smart Public Service CRM provides a centralized platform where citizens can:

- Submit complaints or service requests.
- Track request status in real time.
- View SLA deadlines for resolution.

Government employees can manage requests through a workflow system, while administrators monitor performance through dashboards.

**Key Features**

- Citizen dashboard with service and complaint submission.
- Real-time request tracking (Activity page).
- SLA-based request deadlines.
- Employee task assignment system.
- Admin monitoring dashboard.
- FastAPI backend for request management.


**How to Run the Project**
1. Install dependencies
- pip install fastapi uvicorn
2. Start backend server
- python -m uvicorn backend.main:app --reload
3. Open frontend
- frontend/index.html

**Demo Workflow**

- Citizen submits complaint →
- Backend processes request →
- Activity page tracks request status →
- Employee resolves issue →
- Admin monitors system performance.
